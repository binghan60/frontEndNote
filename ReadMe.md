# **一、掃描 QRCode、BarCode**

## 1.Html5-QRCode<br>

同時掃描多種樣式的 QRCode<br>
但無法在 Liff 開啟<br>
[Html5-QRCode 官方文件](https://scanapp.org/html5-qrcode-docs)<br>

## 2.quaggaJS<br>

掃描 BarCode<br>
一次只能指定某種規格 BarCode<br>
可以在 Liff 開啟<br>
[quaggaJS 官方文件](https://serratus.github.io/quaggaJS)<br>

也可以搭配框架<br>
[Electron 官方文件](https://www.electronjs.org/)<br>
